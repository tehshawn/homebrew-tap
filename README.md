# homebrew-tap

Homebrew tap for [tehshawn](https://github.com/tehshawn)'s apps.

## MKV QuickPlay

A lightweight macOS menu bar app for quick video preview using mpv.

```bash
brew install --cask tehshawn/tap/mkvquickplay
```

This installs the notarized [MKV QuickPlay](https://github.com/tehshawn/mkvquickplay) app and its `mpv` dependency. To update:

```bash
brew upgrade --cask mkvquickplay
```
